| フェーズ                                     | 用途                         |
| -------------------------------------------- | ---------------------------- |
| [pre-release](./pre-release/index.md)        | 新規開発（初回リリースまで） |

| [common/env-setup.md](./common/env-setup.md) | 環境整備（共通）             |
